# Copilot Chat

1. Ask：コメントの表示
2. Edit:コメントの追加
3. Agent
    1. バグ埋め込んでテストして直す
    2. ブランチ作成・コミット・プッシュ
4. カスタムチャットモード
    1. 掛け算の計画を立てる
    2. IssueをMCPを使って建てる
