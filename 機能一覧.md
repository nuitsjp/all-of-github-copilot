# GitHub Copilot 機能一覧（2025年1月時点）

## GitHub Copilot機能

| 機能名 | 概要 | 対応環境 | Pro | Pro+ | Business | Enterprise |
|--------|------|----------|-----|------|----------|------------|
| Code completion | リアルタイムでの自動補完スタイルのコード提案 | VS Code, Visual Studio, JetBrains, Azure Data Studio, Xcode, Vim/Neovim, Eclipse | ✅ | ✅ | ✅ | ✅ |
| Copilot Chat | コーディング関連の質問ができる対話型インターフェース | GitHub Web, GitHub Mobile, IDE, Windows Terminal | ✅ | ✅ | ✅ | ✅ |
| Copilot coding agent | GitHub IssueからPRまでの自動実装 | GitHub | ❌ | ✅ | ✅ | ✅ |
| Copilot in the CLI | コマンドラインでのコマンド提案・説明 | ターミナル, Windows Terminal Canary | ✅ | ✅ | ✅ | ✅ |
| GitHub Copilot code review | AI生成のコードレビュー提案 | GitHub | ❌ | ✅ | ✅ | ✅ |
| Copilot pull request summaries | PR変更の自動要約生成 | GitHub | ❌ | ✅ | ✅ | ✅ |
| Copilot Workspace | PRの改良・検証・レビュー統合環境 | GitHub | ❌ | ✅ | ✅ | ✅ |
| Copilot text completion | PR説明文の迅速・正確な作成支援 | GitHub | ❌ | ❌ | ❌ | ✅ |
| GitHub Copilot Extensions | 外部ツールのGitHub Copilot Chatへの統合 | GitHub | ✅ | ✅ | ✅ | ✅ |
| GitHub Models | 業界をリードする大規模・小規模言語モデルへの直接アクセス | GitHub | ✅ | ✅ | ✅ | ✅ |
| Copilot Edits | 単一プロンプトから複数ファイルへの変更 | VS Code, Visual Studio, JetBrains | ✅ | ✅ | ✅ | ✅ |
| - Edit mode | 細かな制御での編集提案 | VS Code, JetBrains | ✅ | ✅ | ✅ | ✅ |
| - Agent mode | 自律的なコード編集・タスク完了 | VS Code, Visual Studio, JetBrains | ✅ | ✅ | ✅ | ✅ |
| Copilot custom instructions | 個人の好み・ツール・要件に基づく応答カスタマイズ | GitHub, IDE | ✅ | ✅ | ✅ | ✅ |
| Copilot in GitHub Desktop | コミットメッセージの自動生成 | GitHub Desktop | ✅ | ✅ | ✅ | ✅ |
| Copilot Spaces | 関連コンテンツの整理・集中化とコンテキスト共有 | GitHub | ❌ | ✅ | ✅ | ✅ |
| Copilot knowledge bases | 組織ドキュメントのコレクション作成・管理 | GitHub, VS Code | ❌ | ❌ | ❌ | ✅ |

## 管理者向け機能

| 機能名 | 概要 | Business | Enterprise |
|--------|------|----------|------------|
| ポリシー管理 | 組織・エンタープライズでのCopilotポリシー管理 | ✅ | ✅ |
| アクセス管理 | 組織・エンタープライズメンバーのアクセス制御 | ✅ | ✅ |
| 使用データ | Copilot使用状況の確認・分析 | ✅ | ✅ |
| 監査ログ | Copilotアクションの監査ログ確認 | ✅ | ✅ |
| ファイルの除外 | 特定ファイルのCopilotからの除外設定 | ✅ | ✅ |
