## Copilot Chat モードの詳細

### Ask モード
- **用途**: 質問・説明・理解
- **特徴**: コードの説明、技術的な質問、ブレインストーミング
- **コンテキスト**: #file、#codebase、#selection など

### Edit モード
- **用途**: 複数ファイル編集
- **特徴**: 制御された編集、ファイル指定、段階的な変更
- **作業セット**: 明示的にファイルを指定して編集

### Agent モード
- **用途**: 自律的タスク実行
- **特徴**: 複雑なタスク、ツール実行、反復的な問題解決
- **ツール**: ファイル操作、ターミナル実行、エラー修正、テスト実行
